# @lukim9-kakao/client-android

High-level KakaoTalk Android client: auth, bootstrap, channel API, and AndroidTalkClient

Part of the **node-kakao v5** workspace — a modern rewrite of the KakaoTalk LOCO client.
See the [repository](https://github.com/Lukim99/node-kakao-stable) for full docs and the
high-level `AndroidTalkClient`.

Prerelease (`5.0.0-alpha.0`): install with the `alpha` tag.

```sh
npm install @lukim9-kakao/client-android@alpha
```

Derived from the original [node-kakao](https://github.com/storycraft/node-kakao) (MIT).
